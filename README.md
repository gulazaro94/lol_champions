# lol_champions

## Requirements

- Docker
- Docker compose
- lol API key

## Running

    $ git clone https://github.com/gulazaro94/lol_champions.git
    $ cd lol_champions
    $ echo "API_KEY=<your_api_key>" > .env
    $ docker-compose build
    $ docker-compose up

Now access the docker host on port 9292. Ex: http://localhost:9292
